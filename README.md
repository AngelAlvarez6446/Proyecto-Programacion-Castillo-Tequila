## Interfaz de Inteligencia - El Castillo de Tequila

Este proyecto tiene como objetivo diseñar una herramienta de análisis de datos para El Castillo de Tequila, 
permitiendo recolectar, centralizar y analizar la información obtenida a través de encuestas aplicadas a los visitantes.

El sistema está desarrollado en Python utilizando Programación Orientada a Objetos (POO), e incluye:

- Lectura de respuestas desde Google Sheets (exportadas como CSV).
- Modelado de visitantes utilizando clases, herencia y polimorfismo.
- Almacenamiento de datos en una base de datos SQLite.
- Módulo de analítica para métricas de mercado, engagement y correlación de opiniones.

## Requisitos

- Librerías:

  - numpy
